# vue-v-directives
Vue custom directive extensions, including long press, copy, watermark, permissions and more.

## 安装
```bash
npm install --save vue-v-directives
```
## 引用
```javascript
import Vue from 'vue'
import directives from 'vue-v-directives'
Vue.use(directives)
```
