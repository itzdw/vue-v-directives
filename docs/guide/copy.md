# 文本复制 copy
绑定该指令，点击元素才能实现复制功能

## 正常用法
```vue
<template>
  <textarea v-model="content"></textarea>
  <br>
  <button v-copy="success" :copy-text="content">文本复制</button>
</template>

<script>
export default {
  data() {
    return {
      content: 'Hello directives!'
    }
  },
  methods: {
    success(val) {
      alert('复制成功，内容为：' + val)
    }
  }
}
</script>
```
